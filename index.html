HTML (Hypertext Markup Language) is a markup language used to create and structure content for the web. HTML uses tags and attributes to define the structure and formatting of web pages, which are then rendered by web browsers.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of web pages written in HTML or XML. CSS allows web developers to separate the visual style of a web page from its structure and content, making it easier to maintain and update the design of a website.

Bootstrap

Bootstrap is a popular front-end web development framework that helps developers create responsive and mobile-first websites and web applications. Bootstrap provides a set of pre-designed HTML, CSS, and JavaScript components, such as buttons, forms, modals, and navigation menus, that can be easily customized and integrated into web projects.Bootstrap was developed by Twitter and released as an open-source project in 2011. It is now maintained by a community of developers and has become one of the most widely used front-end frameworks.

Ceiling Air Purifier

An air ceiling purifier is a type of air purifier that is designed to be mounted on the ceiling. It is typically used in commercial or industrial settings, but can also be found in some residential settings.The purpose of an air ceiling purifier is to remove pollutants and contaminants from the air, including dust, allergens, bacteria, and viruses. It works by drawing in air from the surrounding environment, filtering it through a series of air filters, and then returning the purified air back into the room.
